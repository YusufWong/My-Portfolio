3D Printed Elbow Exoskeleton

This project focuses on designing and fabricating a 3D-printed elbow exoskeleton to assist individuals with mobility impairments.

Purpose

To design an exoskeleton using SolidWorks to generate concept designs and assemblies, and learn mechatronics concepts including Arduino programming, actuator control, and biomechanics.

Images

Challenges

While testing the load cell, the hinge broke at the motor connection interface. Temporary fixes were applied using tape and screws.

Learnings

  • 3D modeling robotic assemblies
  • Understanding control systems, load cell sensors, and PID control
  • Arduino programming and debugging techniques
